# Teletext Radio

Teletext Radio is a synchronized 24/7 online radio streaming platform that ensures all listeners hear the exact same content at the same time. Built entirely with vanilla JavaScript, it provides a seamless streaming experience with time-synchronized playback across all connected clients.

## What makes Teletext Radio unique?

Teletext Radio implements a sophisticated synchronization system that calculates the exact position in the playlist based on elapsed time since transmission started. This means everyone listening to the radio hears the same track at the same second, creating a truly shared listening experience.

## Key features

### Synchronized playback

The radio uses a deterministic algorithm to ensure all listeners are perfectly synchronized. By calculating elapsed time from a fixed transmission start date, the system determines exactly which track should be playing and at what timestamp.

```javascript
function calcularPosicionExacta() {
    const inicioTransmision = new Date('2025-01-01T03:00:00Z');
    const ahora = new Date();
    
    const segundosTranscurridos = Math.floor((ahora - inicioTransmision) / 1000);
    const segundosPorCancion = 180;
    const segundosTotalPlaylist = currentPlaylist.length * segundosPorCancion;
    const posicionEnPlaylist = segundosTranscurridos % segundosTotalPlaylist;
    
    currentTrackIndex = Math.floor(posicionEnPlaylist / segundosPorCancion) % currentPlaylist.length;
    const segundoEnCancion = posicionEnPlaylist % segundosPorCancion;
    
    return {
        trackIndex: currentTrackIndex,
        segundoEnCancion: segundoEnCancion,
        track: currentPlaylist[currentTrackIndex]
    };
}
```

### Dynamic schedule management

Teletext Radio features a 24/7 programming schedule with different shows throughout the day. The schedule automatically updates based on Argentina time (UTC-3) and displays the current program.

**Daily schedule:**
- **Radio 404** - 01:00 to 06:00 - Atmospheric and experimental sounds
- **Archivo txt** - 06:00 to 12:00 - Morning energy and rhythms
- **Telesoft** - 12:00 to 16:00 - Varied afternoon selections
- **Floppy Disk** - 16:00 to 20:00 - Evening atmospheric transitions
- **Internet Archive** - 20:00 to 01:00 - Extended night sessions
- **Especiales txt** - 22:00 to 00:00 (Friday & Saturday) - Special weekend programming

### Playlist-based streaming

The platform loads tracks from a `playlist.json` file, making it easy to update your music library without touching the code:

```json
{ "tracks": [
  "music/andresnewforu.mp3",
  "music/archivotxt/anaoxacontrolo.mp3",
  "music/archivotxt/bizarretriangule.mp3"
] }
```

### Responsive player interface

The fixed player stays at the top of the page and displays:
- Station branding with custom font (Chomsky)
- Live status indicator with pulsing animation
- Current show name
- Play/pause controls
- Share button to copy the radio URL

<Note>
The player interface is fully responsive and works seamlessly on desktop, tablet, and mobile devices.
</Note>

## Technology stack

Teletext Radio is built with:

- **Vanilla JavaScript** - No frameworks, pure JavaScript
- **HTML5 Audio API** - Native browser audio playback
- **CSS3** - Modern responsive styling with backdrop filters and animations
- **JSON** - Simple playlist and schedule configuration
